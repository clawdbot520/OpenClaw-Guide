# 2.1 精品玻璃景觀缸：Mac 玩家實戰手冊 🦞

在 Mac 上佈置你的第一座數位魚缸是一件既優雅又高效的事。本節將根據官方推薦的標準工法，指導你一步步完成環境建置，確保你的龍蝦特務能在最穩定的水域中誕生。

---

### 一、 建置前準備

在開始安裝前，請確認你的 Mac 符合以下核心硬體規格：
*   **處理器**：Apple Silicon (M1/M2/M3/M4) 系列效果最佳，Intel 系列同樣支援。
*   **作業系統**：建議使用最新版本的 macOS。
*   **開發套件**：Mac 需要先安裝基礎開發工具，請在終端機輸入：
```bash
   xcode-select --install
```
這能確保後續安裝過程不會遇到管線堵塞。

---

### 二、 核心安裝步驟

#### 1. 安裝執行環境：Node.js
OpenClaw 對大腦活力的要求很高，請務必安裝 **Node.js v22** 或更高版本。推薦使用 Homebrew 進行安裝：
```bash
brew install node
```
*檢查版本：* 輸入 `node -v`，確認數字是大於或等於 22。

#### 2. 下載並安裝 OpenClaw CLI
這一步會將小龍蝦特務的核心管理工具安裝到你的全域環境中：
```bash
npm install -g openclaw@latest
```
*檢查身分：* 輸入 `openclaw --version`，如果看見版本編號，代表特務已經整裝待發。

#### 3. 初始化：點亮魚缸
現在，我們要執行最重要的初始化指令。這不只是設定，更像是為特務進行一次全方位的身體檢查與呼吸訓練：
```bash
openclaw onboard --install-daemon
```

**【新手必讀】導航介面詳細解析：**

這是一個對話式的導航介面，系統會一步步詢問你，請按照以下邏輯進行選擇：

*   **選擇通訊頻道 (Channels)**：
    系統會問你想要在那裡跟龍蝦聊天。對於新手，雖然大家最常用Line，但考量到安全性，怕誤操作你的Line設定，我強烈建議選擇 **Telegram**。同時它不需要繁瑣的伺服器設定，只要填入你從 BotFather 拿到的 Token 即可（詳見第三節）。
*   **配置核心模型 (Model Configuration)**：
    這是最關鍵的一步。系統會列出多個供應商，以下是我的專業配方建議：
    *   **追求穩定與中文能力**：請選擇 **MiniMax** 或 **DeepSeek**。它們對繁體中文的理解極佳，且 API 申請門檻低、價格相對實惠。
    *   **追求頂尖智慧**：選擇 **Claude (Anthropic)** 或 **OpenAI**。如果你的任務涉及複雜的程式撰寫或逻辑推理，這是首選。
    *   **極致隱私與免費**：如果你已經安裝了 **Ollama**，請選擇「Local Ollama」。這會讓龍蝦完全跑在你的電腦內，不花一毛錢。
*   **設定認證模式 (Auth Mode)**：
    系統會詢問門禁機制。請選擇 **password**（設定一個你自己記得住的密碼），這樣你以後開啟網頁管理介面時，才不會被擋在門外。
*   **安裝守護進程 (Daemon)**：
    因為我們加了 `--install-daemon` 參數，系統會自動在背景安裝維生系統。這意味著即使你關掉這台電腦的螢幕，龍蝦依然會在數位大海中為你巡邏。

---

### 三、 配置對講機：Telegram Bot

在執行 `onboard` 過程中，系統會詢問你要接入哪個通訊頻道。建議首選 Telegram：
1.  開啟 Telegram，搜尋 **@BotFather**。
![BotFather Guide](assets/Screenshot%202026-03-12%20at%208.41.56%20PM.png)
2.  輸入 `/newbot`，獲得專屬的 **API Token**。
3.  將這串金鑰輸入到 `onboard` 的提示框中。

---

### 四、 避雷針與養殖建議

{% hint style="warning" %}
**專業提示**
*   **認證模式設定**：由於系統版本更新，現在啟動時必須明確設定認證模式。建議在首次設定時選定為 `token` 或 `password` 模式，以確保魚缸安全。
*   **獨立工作空間**：建議將特務的工作區設定在 `~/.openclaw/workspace`，保持環境整潔。
{% endhint %}
