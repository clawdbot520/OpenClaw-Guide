# 2.2 都市公寓的空間魔法：Windows 玩家實戰手冊 🦞

在 Windows 上養殖龍蝦特務，最好的方式並非直接安裝在系統表面，而是利用黑科技 **WSL2** 打造一個如同天然水域的執行環境。

官方強烈建議 Windows 玩家透過 Linux 子系統運行 OpenClaw，這能避開絕大多數的路徑、權限與性能瓶頸，讓你的特務游得更順暢。

---

### 一、 開啟傳送門：安裝 WSL2

首先，我們需要讓你的 Windows 具備運行 Linux 的能力。請開啟 **PowerShell**（以系統管理員身分執行），輸入：
```powershell
wsl --install
```
系統會自動準備好環境並預設安裝 **Ubuntu**。完成後，請務必重啟電腦。

---

### 二、 核心安裝步驟

接下來的所有操作，都請在重啟後開啟的 **Ubuntu 終端機** 視窗中進行。

#### 1. 更新系統水質
在開始安裝前，先確保環境是最新的：
```bash
sudo apt update && sudo apt upgrade -y
```

#### 2. 安裝執行環境：Node.js
我們需要 **Node.js v22** 或更高版本。請使用以下方式精準安裝：
```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```
*檢查版本：* 輸入 `node -v` 確保版本號為 22 以上。

#### 3. 下載並安裝 OpenClaw CLI
這會將特務的核心能力安裝到你的 Ubuntu 地盤：
```bash
npm install -g openclaw@latest
```

#### 4. 初始化：點亮魔法空間
輸入最重要的引導指令，按照畫面提示完成特務的身份與能力設定：
```bash
openclaw onboard --install-daemon
```

**【新手必讀】導航介面詳細解析：**

在 Ubuntu 終端機中，系統會透過問答方式帶領你完成配置：

*   **選擇通訊渠道 (Channels)**：
    強烈建議新手先選擇 **Telegram**。你只需要提供 Bot Token，不需要處理複雜的網路通訊埠轉發。
*   **配置核心模型 (Model Configuration)**：
    *   **追求穩定與中文能力**：請選擇 **MiniMax** 或 **DeepSeek**。這對台灣玩家來說非常親切，且設定過程極簡。
    *   **追求頂尖智慧**：選擇 **Claude (Anthropic)** 或 **OpenAI**。
    *   **本地運行**：如果你已在 WSL2 內安裝了 **Ollama**，可直接連動。
*   **設定認證模式 (Auth Mode)**：
    這步非常重要！請務必選擇 **password** 並設定一組門禁密碼。因為 WSL2 運行的服務在 Windows 端通常透過 `localhost` 訪問，設定密碼能確保你的 Web UI 不會被他人隨意操控。
*   **守護進程 (Daemon)**：
    指令中的 `--install-daemon` 會確保龍蝦在背景運作，即使你關閉 Ubuntu 視窗，特務依然在線。

---

### 三、 專業遙控：VS Code 連動

雖然特務住在 Linux 湖泊裡，但你可以坐在 Windows 辦公桌前進行遠端管理：
1.  安裝 Windows 版的 **VS Code**。
2.  在擴充商店搜尋並安裝 **WSL** 元件。
3.  在 Ubuntu 視窗中輸入 `code .`，就能像操作本機檔案一樣，優雅地調整龍蝦的 `openclaw.json` 設定檔。

---

### 四、 Windows 玩家避雷手冊

{% hint style="warning" %}
**安全提醒**
*   **檔案路徑原則**：請將你的工作區目錄設定在 Ubuntu 自己的檔案系統內（例如：`~/openclaw-work/`），不要放在映射的 `C:/` 槽，這能避免特務在處理資料時產生嚴重的延遲。
*   **認證模式強制要求**：自版本 v2026.3.7 起，啟動時必須在設定中明確指定認證模式。如果你想透過網頁介面管理，請務必設定好密碼或權杖。
{% endhint %}
