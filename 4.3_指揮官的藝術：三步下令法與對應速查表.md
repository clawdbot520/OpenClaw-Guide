# 4.3 指揮官的藝術：三步下令法與對應速查表 🦞

當你已經設定好了四大魂核文件，你的龍蝦特務已經具備了極佳的「底子」。接下來，就看你這位「指揮官」如何活用心法（結構）與招式（速查表）來下達下達指令了。

### 必勝心法：三步下令法

不要只是丟出一句模糊的命令，試著套用這個黃金公式：

1.  **Role (設定角色)**：先告訴龍蝦牠現在是誰。「你是我的高級情報分析官。」
2.  **Context (提供背景)**：給予必要的資訊與限制。「我要分析這集 Podcast 的音檔，我只需要關於投資的部分。」
3.  **Goal (明確目標)**：定出輸出的規格。「請列出 3 個核心觀點，並用條列方式呈現。」

當你把 **角色、背景、目標** 三位一體地交給龍蝦，牠就不會像沒頭蒼蠅一樣亂撞，而是會像精準的特務一樣，直接切中要害。

### 實戰招式：特務語轉換速查表

為了幫你快速填補人學與特務邏輯之間的鴻溝，我們整理了這張表格。當你想說「人話」時，請試著在心裡進行轉換：

| 當你想說（人話口語） | 請改說（特務指令） | 為什麼這樣更強大？ |
| :--- | :--- | :--- |
| **幫我想想...** | **請分析現狀並提供三個具備可行性的範例。** | 給出「數量」與「可行性」約束。 |
| **寫下來。** | **請將結果輸出為 Markdown 格式的條列清單。** | 明確的規格方便你後續複製存檔。 |
| **查一下。** | **請使用搜尋技能，並總結出三個核心結論。** | 告知動作（搜尋）與產出（總結）。 |
| **隨便寫一點。** | **請根據上述背景，草擬一段約兩百字的短文。** | 去除模糊詞，精確控制耗費的 **Token**。 |
| **有什麼建議？** | **請從財務與時間成本兩個角度，評估此決策的優劣。** | 強制特務從特定維度分析，建議才具參考價值。 |

### 指揮官的高級技巧：範例與約束

*   **給予範例 (One-Shot Prompting)**：
    如果你想要特務寫出某種特定風格的文章，與其解釋一百次規則，不如直接丟給牠一個你喜歡的樣版說：「請模仿這個風格來撰寫」。這能瞬間拉高特務的表現上限。
*   **鎖定動詞**：
    盡量避免使用「處理」、「搞定」這種含糊的動詞。改用「提取」、「摘要」、「比較」、「轉譯」等具備明確邏輯的詞彙。

### 結語：你是特務的北極星

在人機共生的路上，你是提供願景的人，而龍蝦是幫你抵達終點的船。掌握了結構與速查表，你就不再是那個坐在翻車博物館裡抱怨的人，而是一個優雅的數位指揮官。

**指令優化終極心法**
*   **具體大於模糊**：拒絕「隨便」、「儘快」。
*   **精確動詞**：動詞越精確，特務的搜尋空間就越窄。
*   **保持耐心**：把溝通落差當成優化指令的「反饋」，而非能力的失敗。
