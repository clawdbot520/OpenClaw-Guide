# 5.3 全球武器販賣商店：去市場挑選最強戰種 🦞

如果內建的技能已經無法滿足你與特務的雄心壯志，那麼是時候走出魚缸，去 **ClawHub** 全球武器販賣商店逛逛了。

在 OpenClaw 的世界裡，你不需要自己從零開始編寫程式來教龍蝦新功能。全球有成千上萬的開發者正在不斷編寫新的 **skill** 並上傳至這個「地下軍火庫」。你只需要知道如何領取並安裝這些「高階外掛」，你的特務就能瞬間進化。

### 點亮你的技能樹

安裝新技能的過程，就像是在角色扮演遊戲中點亮天賦樹，或者是在特務電影中挑選最新科技工具：

*   **搜尋裝備 (Search)**：
    不確定有什麼好東西？直接問商店：
    ```bash
    openclaw skills search <關鍵字>
    ```
    系統會列出 ClawHub 中所有相符的裝備（Skills）與簡介。你也可以直接訪問官網 `hub.openclaw.io` 查看熱門排行與圖文介紹。

*   **查看詳情 (Info)**：
    安裝前想看說明書？執行：
    ```bash
    openclaw skills info <技能名稱>
    ```

*   **一鍵安裝 (Install)**：
    選好裝備後，在你的終端機輸入：
    ```bash
    openclaw skills install <技能名稱>
    ```
    系統會自動從全球倉庫抓取裝備。完成後，特務就會發現自己的戰術背包裡多了一樣超酷的新功能。

### 什麼樣的裝備值得下載？

在商店貨架上，你可以找到各種千奇百怪但實用至極的工具：
*   **財務雷達**：自動抓取主流交易平台的即時幣價。
*   **影迷管家**：幫你追蹤最愛的影集更新進度，並在你下班前提醒你。
*   **學霸義肢**：自動將長篇的學術 PDF 轉化為易懂的圖表與重點摘要。

### 如果找不到合適的裝備怎麼辦？

這就是 OpenClaw 最迷人的地方——**自我擴展的能力**。

如果你需要的裝備在商店裡找不到，你甚至可以直接對特務下達指令：「我需要一個能幫我檢查這份 Excel 資料異常的技能，你能幫我寫一個嗎？」

聰明的特務會運用牠的程式開發能力，嘗試在你的電腦環境中寫出一段全新的腳本，測試無誤後將其封裝成一個專屬於你的 **skill**。這就像是你的特務在戰場上撿到零組件，現場組裝出一把能解決燃眉之急的新武器。

### 安全養殖警語

雖然公會裡的裝備琳瑯滿目，但請記住，不是所有的龍蝦都是善意的。在安裝來路不明的社區技能前，請務必確認其評價，或詢問資深養殖者的建議。確保你的魚缸（環境）有良好的隔離措施，別讓惡意的「外來魚種」破壞了你的數位生態。
